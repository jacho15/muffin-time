# Muffin Time

A productivity app that combines calendar management, task tracking, and focused study timers.

## Overview

**Muffin Time** aims to centralize your workflow into a single, immersive environment.

## Key Features

- **Multi-Calendar Support**: View and manage your schedule with a fluid, week/month grid.
- **Notion-Style Tasks**: Organize your life with a powerful task management system.
- **Orbital Study Timer**: Stay focused with a Pomodoro-inspired timer featuring orbital animations and cosmic backgrounds.
- **Time Insights**: Visualize your productivity with beautiful charts and heatmaps.
- **Cosmic Night Theme**: Immersive UI with stars, nebulae, and drifting shooting stars powered by Framer Motion.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/jacho15/muffin-time.git
   cd muffin-time
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Navigate to `http://localhost:5173` to see the stars!

## Testing

We use [Vitest](https://vitest.dev/) for unit and integration tests.

Run tests once:
```bash
npm run test
```

Start vitest in watch mode:
```bash
npm run test:watch
```
